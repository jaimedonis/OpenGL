This is an OpenGL-based gltf scene renderer. 

For this branch I use of the Android Studio build system.

Features:
- PBR and BSDF shaders
- Simple scene graph
- OpenGL-ES

Upcoming Features:
- TBD, volumetric rendering support.
- Cross-platform
